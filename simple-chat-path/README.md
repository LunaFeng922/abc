## Chat Path

Hi! This project is a chat room visualizes the evolving "chat path" between you and the people you're chatting with —- a "chatting-visualization" of your relationships.

### Installation & Setup

1. **Clone the repository**

2. **Install and run**
   - Open terminal
    - ` cd project`
    - ` npm install`
    - `node server.js`
   - Note the HTTPS server port number from the output.

3. **Access the application**
   - Find your local network IP address
   - Navigate to: `https://your-ip-address:port-number`

### How to Use

1. **Set your name** — Start by giving yourself a username before chatting

2. **Try different messages** to see how they affect the chat path:
   - Type phrases like "love you" to see positive interactions
   - Try phrases like "hate you" to observe negative impacts
